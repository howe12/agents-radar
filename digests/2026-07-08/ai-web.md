## 二、Anthropic / Claude 内容精选

### 📰 News（产品发布）

#### [Introducing Claude Sonnet 5](https://www.anthropic.com/news/claude-sonnet-5)
- **发布日期**：2026-07-07（公告日）；模型版本日期标注 2026-06-30
- **核心要点**：
  1. **Agentic 能力的代际跃迁**：官方宣称 Sonnet 5 是"迄今最具 agentic 能力的 Sonnet 模型"，能够在浏览器、终端等工具环境中自主规划并执行长链路任务——这一能力"在几个月前还只有更大、更昂贵的模型才能做到"。
  2. **性能接近 Opus 4.8，价格仍为 Sonnet 档**：在推理、工具使用、编码、知识工作等关键 agentic 维度上实现对 Sonnet 4.6 的显著提升，综合表现逼近 Opus 4.8，但维持 Sonnet 档定价。这是典型的"性能阶梯下移"策略。
  3. **全线可用，Free 用户默认启用**：Sonnet 5 在所有订阅层级（Free / Pro / Max / Team / Enterprise）同步上线，Free 与 Pro 用户将其设为默认模型——这是 agentic 能力首次对 C 端免费用户开放。
  4. **安全姿态显性化**：System Card 报告称 Sonnet 5 的"不良行为率"低于 Sonnet 4.6，且在网络安全任务上的能力明显弱于当前 Opus 模型——后者是一种刻意的能力边界声明，与 Anthropic 长期强调的"responsible scaling"叙事一致。
